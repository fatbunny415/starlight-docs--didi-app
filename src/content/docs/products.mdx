### Microservicio de Productos

<Aside type="caution" title="IMPORTANTE">
Este servicio permite gestionar el catálogo de productos disponibles en el sistema.  
Es esencial para mantener actualizada la información que consumen los servicios de pedidos y visualización.
</Aside>

El microservicio de **Productos** permite realizar operaciones CRUD sobre los productos registrados en la base de datos.  
Utiliza SQLAlchemy para la persistencia y se integra con otros servicios mediante FastAPI.

## Listar Productos

Obtiene todos los productos registrados en la base de datos.

**endpoint** <Badge text="GET" size="large" variant="caution" />

```
/api/v1/products
```

**Response** <Badge text="200" variant="success" />

```json
[
  {
    "id": 1,
    "cod_general": "P001",
    "name": "Producto A",
    "description": "Descripción del producto A",
    "price": 100.0
  }
]
```

---

## Crear Producto

Registra un nuevo producto en el sistema.

**endpoint** <Badge text="POST" size="large" variant="caution" />

```
/api/v1/products
```

**Request**

```json
{
  "cod_general": "P002",
  "name": "Producto B",
  "description": "Descripción del producto B",
  "price": 150.0
}
```

**Response** <Badge text="201" variant="success" />

```json
{
  "id": 2,
  "cod_general": "P002",
  "name": "Producto B",
  "description": "Descripción del producto B",
  "price": 150.0
}
```

---

## Actualizar Producto

Modifica los datos de un producto existente.

**endpoint** <Badge text="PUT" size="large" variant="caution" />

```
/api/v1/products/{product_id}
```

**Request**

```json
{
  "cod_general": "P002",
  "name": "Producto B Modificado",
  "description": "Nueva descripción",
  "price": 180.0
}
```

**Response** <Badge text="200" variant="success" />

```json
{
  "id": 2,
  "cod_general": "P002",
  "name": "Producto B Modificado",
  "description": "Nueva descripción",
  "price": 180.0
}
```

<Badge text="404" variant="danger" />

```json
{
  "detail": "Producto no encontrado"
}
```

---

## Eliminar Producto

Elimina un producto por su ID.

**endpoint** <Badge text="DELETE" size="large" variant="caution" />

```
/api/v1/products/{product_id}
```

**Response** <Badge text="200" variant="success" />

```json
{
  "message": "Producto eliminado correctamente"
}
```

<Badge text="404" variant="danger" />

```json
{
  "detail": "Producto no encontrado"
}
```
