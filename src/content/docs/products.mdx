### Microservicio de Productos

<Aside type="caution" title="IMPORTANTE">
Este servicio permite gestionar el catálogo de productos disponibles en el sistema.  
Incluye operaciones para listar, crear, actualizar, consultar y eliminar productos, con manejo de errores y validaciones.
</Aside>

El microservicio de **Productos** utiliza FastAPI y SQLAlchemy para la persistencia de datos.  
Se encarga de mantener actualizada la información que consumen otros servicios como pedidos y visualización.

## Listar Productos

**endpoint** <Badge text="GET" size="large" variant="caution" />
```
/api/v1/products
```

**Response** <Badge text="200" variant="success" />
```json
[
  {
    "id": 1,
    "cod_general": "P001",
    "name": "Producto A",
    "description": "Descripción del producto A",
    "price": 100.0
  }
]
```

<Badge text="500" variant="danger" />
```json
{
  "detail": "Error interno del servidor: [detalle técnico]"
}
```

## Crear Producto

**endpoint** <Badge text="POST" size="large" variant="caution" />
```
/api/v1/products
```

**Request**
```json
{
  "cod_general": "P002",
  "name": "Producto B",
  "description": "Descripción del producto B",
  "price": 150.0
}
```

**Response** <Badge text="201" variant="success" />
```json
{
  "id": 2,
  "cod_general": "P002",
  "name": "Producto B",
  "description": "Descripción del producto B",
  "price": 150.0
}
```

<Badge text="400" variant="danger" />
```json
{
  "detail": "Error al crear producto: [detalle técnico]"
}
```

<Badge text="500" variant="danger" />
```json
{
  "detail": "Error interno del servidor: [detalle técnico]"
}
```

## Actualizar Producto

**endpoint** <Badge text="PUT" size="large" variant="caution" />
```
/api/v1/products/{product_id}
```

**Request**
```json
{
  "cod_general": "P002",
  "name": "Producto B Modificado",
  "description": "Nueva descripción",
  "price": 180.0
}
```

**Response** <Badge text="200" variant="success" />
```json
{
  "id": 2,
  "cod_general": "P002",
  "name": "Producto B Modificado",
  "description": "Nueva descripción",
  "price": 180.0
}
```

<Badge text="400" variant="danger" />
```json
{
  "detail": "Error al actualizar producto: [detalle técnico]"
```

<Badge text="404" variant="danger" />

<Badge text="500" variant="danger" />

```json
}
```
